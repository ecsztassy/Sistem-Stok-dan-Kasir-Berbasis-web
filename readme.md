# 📦 Sistem Stok dan Kasir Berbasis Web

Aplikasi **Sistem Stok dan Kasir Berbasis Web** adalah sistem informasi yang dirancang untuk membantu pengelolaan persediaan barang dan transaksi penjualan pada toko secara efisien dan terintegrasi.

Sistem ini dikembangkan sebagai solusi untuk mengatasi pencatatan manual yang rentan kesalahan, serta memberikan kemudahan dalam monitoring stok dan pembuatan laporan secara real-time.

---

## 🚀 Fitur Utama

- 📊 Manajemen Data Barang
- 📥 Stok Masuk
- 📤 Stok Keluar
- 🛒 Transaksi Kasir (Point of Sale)
- 🧾 Cetak Struk
- 📈 Laporan Penjualan
- 📦 Laporan Stok
- 👤 Manajemen User (Admin/Kasir)

---

## 🛠️ Teknologi yang Digunakan

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: (PHP)
- **Database**: MySQL
- **Web Server**: Apache / XAMPP
